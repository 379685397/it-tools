# 背景
- 网站目前默认语言为英文，虽有多语言支持，但存在英文占位/提示/错误文案，默认体验非中文。
- 新需求：默认中文展示，保留语言切换；英文作为备用语言。

# 目标
- 将站点默认 locale 切换为中文，并确保全局提示/错误/标签等文案可用中文呈现。
- 清理常见硬编码英文（选择器占位、标题等），补充中英文翻译。
- 保留语言切换控件，英文可作为备用。

# 范围
- i18n 配置：默认 locale -> zh，确保 fallback 正常。
- UI 文案：语言选择器占位、标题、提示、错误信息等常见位置的硬编码英文改为 i18n。
- 校验：快速自测关键页面（首页/404/About/命令面板/语言选择器）。

# 非目标
- 不重写工具内部业务逻辑；不删除其他语言文件。

# 风险与影响
- 可能遗漏分散的英文字符串；需抽样/搜索常见路径。
- 更改默认语言可能影响缓存用户；需保留切换能力。
