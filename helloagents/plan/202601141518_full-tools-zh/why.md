# 背景
- 现有站点默认语言已切为中文，但 ~90 个工具的字段标签、占位符、提示/错误文案及示例文本仍有英文残留。
- 需求：覆盖全部工具，将可合理本地化的文案和示例改为中文，同时保留语言切换、英文备用。

# 目标
- 全量补齐工具的中英文 i18n：标题、字段标签、占位/提示、错误/校验信息、示例文本/默认值（格式类示例可保留原字符但配中文说明）。
- 保持英文语言包完整，避免功能回退；默认展示中文。

# 范围
- 工具目录 `src/tools/*`：Vue 组件与相关 service/model 中的 UI 文案和示例文本。
- 公共区域：如有工具相关提示/错误在共享组件内，一并补齐。
- 不涉及：业务逻辑/算法调整；不删除英文键。

# 非目标
- 不重写工具功能或数据结构；不删除现有语言包。

# 风险与影响
- 文案分散、数量大，遗漏风险高；需脚本/分批巡检。
- 修改示例可能影响测试快照/文本断言；需注意 e2e/单测文案同步。
